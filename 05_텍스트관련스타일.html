텍스트 관련 스타일(text~)

color : 텍스트 색상을 지정

        선택자{
            color:색상명 | 16진수 | rgb(x,x,x) | rgba(x,x,x,x)
        }

text-decoration: 텍스트에 줄을 긋거나 줄을 없앨 때 사용

        선택자{
            text-decoration: none | underline | overline | line-through
        }

  • 텍스트 영역 위에 줄 긋기

        선택자{
